 # Django Databse Settings

 ```

    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'django_admin_log',
        'USER' : 'root',
        'PASSWORD' : '',
        'HOST' : 'localhost',
        'PORT' : '3306',
    }
}
```
